Product Review APIs

Moderate customer reviews, toggle visibility, and inject staff replies.

Base URL: /api/v1/product-reviews

Endpoint directory

MethodPathDescriptionAuth
GET/api/v1/product-reviewsPaginated filterable review directory with rating, product, and customer filters.Bearer
GET/api/v1/product-reviews/{id}Loads full review details including moderation metadata and attachments.Bearer
POST/api/v1/product-reviewsAllows staff to seed a curated review on behalf of a user.Bearer
PUT/api/v1/product-reviews/{id}Edits an existing review's visibility, rating, or response.Bearer
DELETE/api/v1/product-reviews/{id}Removes a review permanently.Bearer

List reviews GET /api/v1/product-reviews

Paginated filterable review directory with rating, product, and customer filters.

Parameters

NameDescription
pagePage index
sizePage size
productIdFilter by product
ratingMinMinimum rating
ratingMaxMaximum rating

      "id": 511,

      "visible": true
    }
  ],
