Public Storefront APIs

Anonymous endpoints powering the storefront product listings, detail pages, and promotions.

Base URL: /api/v1/public

Endpoint directory

MethodPathDescriptionAuth
GET/api/v1/public/homeReturns hero banners, featured categories, and spotlight collections for the storefront landing page.Public
GET/api/v1/public/productsSearchable storefront catalogue with pagination, sorting, and filter facets.Public
GET/api/v1/public/products/{slug}Returns full PDP payload including pricing, variants, offers, wedges, reviews, and recommendations.Public
GET/api/v1/public/categoriesReturns visible storefront categories including hero imagery.Public
GET/api/v1/public/couponsLists active coupons with display copy and eligible product/category references.Public
GET/api/v1/public/recently-viewedReturns the last viewed products for the current session or authenticated customer.Public

Homepage content blocks GET /api/v1/public/home

Returns hero banners, featured categories, and spotlight collections for the storefront landing page.

Success response

HTTP 200

{
  "hero": {
    "title": "Glow up your space",
    "cta": "Shop lighting"
  },
  "featuredCategories": [
    {
      "name": "Lighting",
      "slug": "lighting"
    }
  ]
}

Error scenarios

No specific error payloads documented.

List public products GET /api/v1/public/products

Searchable storefront catalogue with pagination, sorting, and filter facets.

Parameters

NameDescription
pagePage index
sizePage size
searchName or SKU
categoryCategory slug
sortprice, -price, newest

Success response

HTTP 200

{
  "content": [
    {
      "id": 101,
      "name": "Aurora Lamp",
      "slug": "aurora-lamp",
      "price": 129.0,
      "thumbnail": "https://cdn.example.com/products/aurora/primary.jpg"
    }
  ],
  "page": 0,
  "size": 20,
  "totalElements": 1,
  "totalPages": 1,
  "last": true
}

Error scenarios

No specific error payloads documented.

Get product by slug GET /api/v1/public/products/{slug}

Returns full PDP payload including pricing, variants, offers, wedges, reviews, and recommendations.

Success response

HTTP 200

{
  "id": 101,
  "name": "Aurora Lamp",
  "slug": "aurora-lamp",
  "offers": [
    {
      "code": "NYGLOW10",
      "discount": "10% off"
    }
  ],
  "variants": [
    {
      "sku": "AUR-LAMP-001",
      "attributes": {
        "Color": "Pearl"
      }
    }
  ]
}

Active coupons GET /api/v1/public/coupons

Lists active coupons with display copy and eligible product/category references.
