Wedge & Highlight APIs

Manage reusable marketing wedges surfaced on product detail pages and landing experiences.

Base URL: /api/v1/wedges

Endpoint directory

MethodPathDescriptionAuth
GET/api/v1/wedgesPaginated wedge directory with category filter and visibility flags.Bearer
POST/api/v1/wedgesCreates a marketing wedge with icon, title, and optional long description.Bearer
PUT/api/v1/wedges/{id}Updates wedge metadata, icon, or enabled status.Bearer
DELETE/api/v1/wedges/{id}Deletes a wedge when not referenced by a template.Bearer
POST/api/v1/wedges/assetsUploads an icon asset for reuse across wedges.Bearer

List wedges GET /api/v1/wedges

Paginated wedge directory with category filter and visibility flags.

Parameters

NameDescription
pagePage index
sizePage size
categoryIdOptional category filter

Success response

HTTP 200

{
  "content": [
    {
      "id": 15,
      "name": "Free Delivery",
      "category": "Product Description Wedges",
      "enabled": true
    }
  ],
  "page": 0,
  "size": 20,
  "totalElements": 1,
  "totalPages": 1,
  "last": true
}
