Product & Asset APIs

Administrative product CRUD, variant authoring, and media asset uploads.

Base URL: /api/v1

Endpoint directory

MethodPathDescriptionAuth
GET/api/v1/productsReturns paginated product summaries with search support.Bearer
GET/api/v1/products/{id}Loads the full editable product payload including variants and media.Bearer
POST/api/v1/productsCreates a new catalog entry with variants, pricing, and media associations.Bearer
PUT/api/v1/products/{id}Updates product details, replacing nested collections supplied in the payload.Bearer
DELETE/api/v1/products/{id}Deletes the product and archived variants.Bearer
POST/api/v1/products/assets/{type}Uploads product images or documents. Supported types include primary, gallery, and manual.Bearer
GET/api/v1/products/assets/{type}/{key}Streams a previously uploaded asset. Responses use the stored MIME type when available.Public

List products GET /api/v1/products

Returns paginated product summaries with search support.

Parameters

NameDescription
pagePage index
sizePage size
searchName/SKU filter

Success response

HTTP 200

{
  "content": [
    {
      "id": 101,
      "name": "Aurora Lamp",
      "slug": "aurora-lamp",
      "sku": "AUR-LAMP-001",
      "status": "ACTIVE"
    }
  ],
  "page": 0,
  "size": 20,
  "totalElements": 1,
  "totalPages": 1,
  "last": true
}

Error scenarios

No specific error payloads documented.

Create product POST /api/v1/products

Headers

NameValue
Content-Typeapplication/json

Request body

{
  "name": "Aurora Lamp",
  "slug": "aurora-lamp",
  "sku": "AUR-LAMP-001",
  "brandId": 6,
  "categoryIds": [
    3
  ],
  "pricing": {
    "currency": "USD",
    "price": 129.0
  }

HTTP 201

{
  "id": 101,

Upload product assets POST /api/v1/products/assets/{type}

Uploads product images or documents. Supported types include primary, gallery, and manual.

Headers

NameValue
Content-Typemultipart/form-data

Request body

No body required.

Success response

HTTP 200

[
  {
    "url": "https://cdn.example.com/products/aurora/primary.jpg",
    "key": "products/aurora/primary.jpg",
    "mimeType": "image/jpeg",
    "sizeBytes": 245678
  }
]

Error scenarios

StatusDescriptionExample
HTTP 400No files provided
{
  "error": "Bad Request",
  "message": "Select at least one file to upload"
}
