Shipping & Logistics APIs

Manage shippable regions, granular rate cards, and delivery rules.

Base URL: /api/v1/shipping

Endpoint directory

MethodPathDescriptionAuth
GET/api/v1/shipping/countriesReturns countries configured for fulfilment including enablement flags.Bearer
POST/api/v1/shipping/countriesAdds a new shippable country with optional cash-on-delivery settings.Bearer
PUT/api/v1/shipping/countries/{id}Updates country-level toggles like enablement, COD allowance, and max weight.Bearer
PUT/api/v1/shipping/countries/bulk-settingsApplies shared configuration to multiple countries in one call.Bearer
DELETE/api/v1/shipping/countries/{id}Deletes a country and cascades to contained states/cities if configured.Bearer
GET/api/v1/shipping/countries/{countryId}/statesReturns states/provinces for the specified country.Bearer
POST/api/v1/shipping/countries/{countryId}/statesCreates a state inside the given country.Bearer
PUT/api/v1/shipping/states/{id}Updates state level toggles or surcharge overrides.Bearer
DELETE/api/v1/shipping/states/{id}Deletes a state and optionally cascades to cities.Bearer
GET/api/v1/shipping/states/{stateId}/citiesReturns configured cities for the given state.Bearer
POST/api/v1/shipping/states/{stateId}/citiesCreates a city under the given state including postal code coverage.Bearer
PUT/api/v1/shipping/cities/{id}Updates city level toggles or surcharges.Bearer
DELETE/api/v1/shipping/cities/{id}Deletes a city and removes associated postal codes.Bearer
GET/api/v1/shipping/area-ratesPaginated rate cards scoped by shipping area.Bearer
POST/api/v1/shipping/area-ratesCreates a new shipping rate with thresholds and surcharge rules.Bearer
PUT/api/v1/shipping/area-rates/{id}Updates base rate, thresholds, or surcharges for an area rate.Bearer
DELETE/api/v1/shipping/area-rates/{id}Deletes a rate card.Bearer

List countries GET /api/v1/shipping/countries

Returns countries configured for fulfilment including enablement flags.

Success response

HTTP 200

[
  {
    "id": 1,
    "code": "US",
    "name": "United States",
    "enabled": true
  }
]
