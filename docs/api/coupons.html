Coupon Management APIs

Create, update, and reference discount coupons with product/category targeting.

Base URL: /api/v1/coupons

Endpoint directory

MethodPathDescriptionAuth
GET/api/v1/couponsPaginated coupon list with type, state, discount, and search filters.Bearer
GET/api/v1/coupons/{id}Returns the full coupon definition including eligibility lists.Bearer
POST/api/v1/couponsCreates a coupon with optional product, category, or user targeting.Bearer
PUT/api/v1/coupons/{id}Updates coupon meta, schedule, or eligibility lists.Bearer
DELETE/api/v1/coupons/{id}Deletes a coupon immediately.Bearer
GET/api/v1/coupons/reference/productsSearchable product dropdown used when targeting specific SKUs.Bearer
GET/api/v1/coupons/reference/categoriesSearchable category dropdown for coupon targeting.Bearer
GET/api/v1/coupons/reference/usersSearches verified users when issuing signup coupons to a subset of accounts.Bearer

List coupons GET /api/v1/coupons

Paginated coupon list with type, state, discount, and search filters.

Parameters

NameDescription
pagePage index
sizePage size
typePRODUCT, CART, SIGNUP
stateDraft/Active/Expired
discountTypePERCENTAGE/FLAT
searchSearch term

Success response

HTTP 200

{
  "content": [
    {
      "id": 41,
      "name": "New Year Glow",
      "code": "NYGLOW10",
      "type": "PRODUCT",
      "state": "ACTIVE"
    }
  ],
  "page": 0,
  "size": 20,
  "totalElements": 1,
  "totalPages": 1,
  "last": true
}

Error scenarios

No specific error payloads documented.

Create coupon POST /api/v1/coupons

Creates a coupon with optional product, category, or user targeting.

Headers

NameValue
Content-Typeapplication/json

Request body

{
  "name": "New Year Glow",
  "code": "NYGLOW10",
  "type": "PRODUCT",
  "discountType": "PERCENTAGE",
  "discountValue": 10,
  "productIds": [
    101
  ],
  "categoryIds": [
    3
  ],
  "startDate": "2025-01-01T00:00:00Z",
  "endDate": "2025-01-31T23:59:59Z"
}

Success response

HTTP 201

{
  "id": 41,
  "code": "NYGLOW10"
}

Error scenarios

StatusDescriptionExample
HTTP 409Code exists
{
  "error": "Conflict",
  "message": "COUPON_CODE_EXISTS"
}
